# Python语言程序与设计
## 豆瓣电影爬虫
### 功能介绍
	豆瓣网电影的排行榜爬取，转存到数据库并输出到文本文件
### 开发环境
	操作系统：windows7 开发语言：python IDE：Pycharm 数据库：mysql
### 数据库表设计
	表名：douban
	列：ID 自增
	列：name （用来存储趴下来的电影）
